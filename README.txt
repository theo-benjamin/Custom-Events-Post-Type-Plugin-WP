== Task ==

Create a Wordpress plugin that adds a custom post type named ‘Events’. The post type should
have the fields ‘Name’, ‘Description’, ‘Sign Up Link’, ‘Start Time’ and ‘End Time’. The plugin
should also allow you to add a shortcode to a page that displays a table with all of the Events.
You can use any additional plugins that you need, as long as you list them as requirements for
your plugin to work.

== Task Breakdown ==



== Installation ==

1. Clone repository and zip file
2.Upload zipfile to wordpress add plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Custom Post Type Event appears at the left menu admin bar  
4. Add shortcode [SpliceEvents] to show events table on front end
